Sublime Text 2 Packages

Curated by Greg

What is Sublime Text 2

Sublime Text 2 is an awesome text editor. A lot of people in the web development community have been adopting it. If you haven't tried it out, you should!

What are Packages

Installing Sublime Package Control is a must if you use Sublime. It lets you install packages (plugins) for Sublime from a huge list of awesome options. The only problem is that there are so many of them!

What is this list

There are a ton of packages for Sublime, and it's time-consuming to try them and decide which you like, especially if there are multiple that do similar things. This page is a list of my personal favorites.

How do I install these

Make sure you have package control installed. Then press cmd-shift-p, type "install package", then press enter. Now type the name of the package you want to install and press enter again. That's it!

GitGutter

GitGutter puts icons in the margin that show you where you've added, modified, or deleted lines according to Git.

SublimeLinter

Inline error-checking for a variety of languages. Shows error message in the status bar.

sftp

Add FTP servers, browse them, open remote files and they'll upload when you save.

git

Everything you can do with the command-line, but in Sublime.

Color Highlighter

Show hex colors when the cursor is on them.

Terminal

Open a terminal at a file's location using the context menu or cmd-shift-t.

View In Browser

View current file in your browser with ctrl-option-v Set your browser in the preferences.

Browser Refresh

Save file and refresh browser with cmd-shift-r.
