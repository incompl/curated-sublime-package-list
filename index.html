Sublime Text 2

Sublime Text 2 is an awesome text editor. A lot of people in the web development community have been adopting it. If you haven't tried it out, you should!

Sublime Package Control

Installing Sublime Package Control is a must if you use Sublime. It lets you install packages (plugins) for Sublime from a huge list of awesome options. The only problem is that there are so many of them!

What is this list

There are a ton of packages for Sublime, and it's time-consuming to try them and decide which you like, especially if there are multiple that do similar things. This page is a list of my personal favorites. Each package has a screen shot that should quickly give you an idea what it does and how it works.
